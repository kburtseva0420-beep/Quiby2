# Быстрый деплой на Railway

## 1. Подготовка

✅ Зарегистрируйтесь на https://railway.app
✅ Создайте GitHub репозиторий с этим кодом
✅ Получите токен бота от @BotFather в Telegram

## 2. Деплой

1. **Railway Dashboard** → "New Project" → "Deploy from GitHub repo"
2. Выберите ваш репозиторий
3. Railway автоматически обнаружит Python и начнёт деплой

## 3. Настройка переменных

В настройках проекта добавьте:

```
BOT_TOKEN=ваш_токен_от_BotFather
```

## 4. Проверка

- Откройте URL проекта (Railway покажет его)
- Должно быть: "Quiz Bot is running!"
- Напишите боту в Telegram `/start`

## Важно!

- `RAILWAY_PUBLIC_DOMAIN` и `PORT` устанавливаются автоматически
- Webhook настраивается автоматически при запуске
- База данных SQLite пересоздаётся при каждом деплое

## Проблемы?

**Бот не отвечает:**
1. Проверьте логи в Railway Dashboard
2. Убедитесь, что `BOT_TOKEN` правильный
3. Проверьте webhook: `https://api.telegram.org/bot<TOKEN>/getWebhookInfo`

**База данных сбрасывается:**
- Это нормально для SQLite на Railway
- Для постоянного хранения используйте PostgreSQL или внешнюю БД
